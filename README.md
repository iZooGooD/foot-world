## FootWorld - Download football highlights free without spoilers

![DEMO](https://s4.gifyu.com/images/UIDemo.gif)


## How to use ?
- Navigate to root directory
- `npm package` - build the executable
- Navigate to `release\build\win-unpacked`
- Run the executable
